# Contributing Guidelines

## 目的

神経多様性を持つ子どもの認知プロフィール観察データを構造化・共有

## 歓迎する貢献

- 構造化された観察データ（JSON形式）
- スキーマ改善提案
- 分析ツール・可視化スクリプト
- ドキュメント改善

## 禁止事項

- 感情的なコメントや主観的な評価
- 根拠のないアドバイスや推測
- 診断・治療の推奨
- 個人を特定可能な情報の記載

## Good Examples

```markdown
Type-Aパターン（役割終了後の離脱困難）、うちでも観察されました。

[JSONデータ添付]
```

## Bad Examples

```markdown
❌ 「可哀想」「頑張れ」などの感情的コメント
❌ 「療育に行くべき」などの治療推奨
❌ 診断名の断定
❌ 本名や詳細な位置情報
```

## データ形式

- スキーマ: `/schema/observation.json` を参照
- テンプレート: `/templates/observation-prompt.md` を参照
- バリデーション: 提出前に `/tools/validate.py` で検証

## 匿名性ガイドライン

### 必須の匿名化

- **個人名**: 使用禁止（仮名・コードネーム可）
- **年齢**: XyXm形式（例: 5y4m）
- **場所**: 一般化（例: 「都内の保育園」）
- **日時**: 相対表記（例: 「2024年12月頃」）

### 推奨される記載方法

- 観察可能な行動のみ
- 文脈情報の明記
- 環境要因の記録
- 再現性のある記述
